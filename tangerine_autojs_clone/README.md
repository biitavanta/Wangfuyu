# Tangerine Auto.js 本地测试版

这个工程只用来测试 `自加老板` 的本地运行。

## 你直接这样用

1. 把整个 `tangerine_autojs_clone` 文件夹导入 `Auto.js Pro 9`
2. 打开 `main.js`
3. 默认已经切到 `自加老板`
4. 在 `自己添加id` 里一行填一个 id
5. 点 `启动和保存文件点这`

## 现在不会做的事

- 不会请求云端
- 不会接飞书
- 不会走老师的每日数据接口

## 本地文件

- 待写队列：`/sdcard/Tangerine每日老板.txt`
- 已写队列：`/sdcard/Storewrittenpath.txt`
